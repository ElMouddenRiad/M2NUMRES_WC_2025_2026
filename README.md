# M2NUMRES Web Components Workspace

Workspace du cours M2 NUMRES avec plusieurs parties pratiques autour de Web Components et Web Audio API.

## Structure
- `Part3/` : projet principal (lecteur audio complet)
- `PART2_WebAudio/` : exercices Web Audio
- `PART1_API/` : API presets (partie antérieure)

## Démarrage rapide (projet principal)
```bash
cd Part3
npx http-server . -p 8000 -c-1
```
Puis ouvrir `http://localhost:8000/index.html`.
